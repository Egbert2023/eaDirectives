The RegEx Tester can be used to test reg expressions before you have to try them out in your own coding. Call regEx.exec(Text) with modyfier 'g' Call regEx.exec(Text) with modyfier 'i' Call regEx.exec(Text) with modyfier 'm' To use the functionality, the following call is required. The parameter data-modyfier can take the values g, i or m. <div ea-reg-test data-modyfier="g"></div> The directive uses the following code.
