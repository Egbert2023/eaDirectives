Example for eaAddHtml

We add the page "Example of accordeon"

Annotation

The "eaAddHtml" directive is used to implement the separation of the design and content of an application. This is realized by extending the Angular routing process in such a way that everything can be parameter-controlled. All routing processes are directed to a fixed HTML page (computed.html), which loads the desired HTML page from server by the parameters from naviList.json.
