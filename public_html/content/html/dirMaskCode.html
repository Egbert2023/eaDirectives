eaMaskCode

This directive is a helper for show code as developer know it from Developer Tools. You can put the code '<any-tag>...</any-tag>' into the tag:
 <ea-mask-code data-code-type="html">
    any-code
 </ea-mask-code>
The field "data-code-type" in the tag can use different values.
  • html - for Html code
  • js - for JavaScript code
  • json - for JSON file
  • css - for CSS file
In the right top corner is a copy button to put the code into the clipboard.
