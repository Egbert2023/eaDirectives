Cookie Policy

Cookies are not used in this application.

This means that neither so-called "technically necessary cookies", no external cookies (e.g. as with Facebook-Like and the like) nor "tracking cookies" (e.g. as with Google Analytics) are set or read on this website. The only exception is the dirCookies demo page. Here the cookies can be switched on and off again explicitly for demo purposes.

External links are marked red, bold and underlined. E.g.: www.externerLink.de If you click on such an external link that leaves your own web application, it will happen in most cases that the respective providers in turn use cookies. The website operator has no influence on this. Please note that you can then receive and edit the corresponding cookie banners to adapt them to your needs, if necessary, in order to protect your privacy.

How to check if no cookies have been set?

Crome-Browser

  • Call the page to be checked (on the example: www.aleksander.de)
  • Press the F12 key
  • In the new window that opens, click on the Application tab.
  • Then select the "Storage" area on the left side
  • In the Storage area there is a sub-item: "Cookies". Select this item.
  • Now you can select the URL to be checked and an overview of existing cookies will appear.

Firefox-Browser

  • Call the page to be checked (on the example: www.aleksander.de)
  • Press the F12 key
  • In the newly opened window click on the tab: "Web storage".
  • Then select the "Cookies" section on the left side. Select it.
  • Now you can select the URL to be checked and an overview of existing cookies will appear.

Edge-Browser

  • Call the page to be checked (on the example: www.aleksander.de)
  • Press the F12 key
  • In the newly opened window click on the "Application" tab.
  • Then select the "Cookies" section on the left and unfold it.
  • Now you can select the URL to be checked and an overview of existing cookies will appear.
